# forRest

**캠핑 장비 중고거래, 보관 및 대여를 제공하는 통합 플랫폼**

요구사항 정의부터 ERD 설계, 구현까지 경험한 Spring Boot 기반 Java Web 프로젝트입니다.
<br>

부피가 큰 캠핑 용품의 보관 문제를 해결하기 위해 캠핑 장비를 보관할 수 있는 장소를 제공하고

보관한 장비를 대여 가능하게 하여 수익을 창출할 수 있습니다.

<br>

## 1. 제작 기간 & 참여인원

* 2022.05.01 ~ 2022.06.30
* 팀프로젝트: 6명

<br>

## 2. 사용 기술

**Back-end**

* java 8
* Spring Boot 2.6.8
* Gradle
* MyBatis
* MariaDB
* Spring Security

**Front-end**

* HTML
* CSS
* Javascript
* JQuery
* Thymeleaf
* ajax
